# LinearAlgebraCourse
This repository contains content for a course in linear algebra.

The content consists of Jupyter notebooks created in Visual Studio Code. The notebooks run in a conda environment. To set up an environment, use the provided `requirements.txt` or `environment.yml` files to install the necessary Python packages.

Python can be installed from https://python.org

Miniconda can be installed from https://docs.conda.io/en/latest/miniconda.html

Visual Studio Code can be installed from https://code.visualstudio.com/
